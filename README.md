# PHP Framework

## $ intro
This is a small php framework to creat a quick web page using php and Boostrap Framework.

## $ Functionality

**Initializing a page**

`$pageName = new Page("The title of page");`

**Theme Light or Dark**\
    `$pageName->theme(""); // light or dark`

**Body Area**
```
$pageName->body("BEGIN");
    //content
$pageName->body("END");
```
